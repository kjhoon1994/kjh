table 태그

예제

표 3-9 테이블 태그
태그 설명
table 표(Table)삽입
thead 표의 머리(Table Head)생성
thead 표의 머리(Table Head)생성
tbody 표의 본문(Table Body)삽입
tfoot 표의 꼬리(Table Foot)생성
tbody 표의 본문(Table Body)삽입
tr 표의 행(Table Row)삽입
th 표의 제목 요소(Table Header)삽입
td 표의 일반 요소(Table Data)삽입

설명

    <table style="width:100%" border = "1">
      <caption>Monthly savings</caption>
      <tr>
        <th>Month</th>
        <th>Savings</th>
      </tr>
      <tr>
        <td>January</td>
        <td>$100</td>
      </tr>
      <tr>
        <td>February</td>
        <td>$50</td>
      </tr>
    </table>
